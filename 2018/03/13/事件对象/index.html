事件对象Event

发表于 2018-03-13

事件对象

定义：再触发DOM上的某个事件，会产生一个事件对象event，这个对象包含所有与事件有关的信息。

DOM上的事件对象

只有在事件执行期间，event对象才会存在，一但执行完毕，event对象就被销毁。

DOM级

兼容DOM的浏览器会将一个event的对象传入到事件处理程序中。无论指定事件处理程序是DOM0级还是DOM2级。

function fn(){
   alert(this);   //这里会出现作用域的问题，在IE中，为window；其他为element
}
var oBn = document.geyElementById("oinput");
oBn.onclick = function fn(event){
        alert(event.type);  //click   
      }
oBn.addEventListener('click', function(event){
           alert(event.type);   //click
},false);

html特性指定

变量event保存着event对象。

<input type='button' value='' onclick='alert(event.type)'/>  //click

event对象所共有的属性和方法

属性 类型 意义
bubbles Boolean 表示事件是否冒泡
cancelable Boolean 表示是否可以取消事件默认事件
currentTarget Element 其事件处理程序当前正在处理事件的元素
defaultPrevented Boolean 为true表示已经调用了preventDefault
detail Element 与事件相关的细节
eventPhase Integer 调用事件处理程序的阶段；1表示捕获阶段，2表示处于目标阶段，3表示冒泡阶段
preventDefault Function 取消事件的默认行为。前提是cancelable为true
stopImmediatePropagetion Function 取消事件的进一步捕获或冒泡，同时阻止任何事件处理程序调用
stopPropagetion Function 取消事件的进一步捕获或冒泡，前提是bubble为true
target Element 事件目标
trusted Boolean 为true表示事件是浏览器生成的，为false表示事件是由开发人员通过js创建的
type string 被触发事件的类型

currentTarget和target

在事件处理程序中this始终等于currentTarget的值，而target则只是事件的真实目标

阻止事件冒泡 

preventDefault()方法

var oBn = document.getElementById('oinput');
oBn.onclick = function(evevt) {
   //若event.cancelable返回true才能执行阻止事件,一般做判断，代码中可不出现
   if(event.cancelable){
        event.preventDefault();
   }
}

 stopPropagetion（）方法

var oBn = document.getElementById('oinput');
oBn.onclick = function(evevt) {
   alert("执行我后并停止向上冒泡！");
   //若event.bubble返回true才能执行阻止事件,一般做判断，代码中可不出现
   if(event.bubble){
        event.stopPropagetion();
   }
}

IE中事件对象

要访问IE事件处理程序的事件对象有几种不同的方法，取决于指定事件处理程序的方法。

DOM0级

//event对象作为window的一个属性

var oBn = document.geyElementById("oinput");
oBn.onclick = function fn(){
        var event = window.event;
        alert(event.type);  //click   
      }

IE事件处理程序

//法一：参数传递

oBn.attachEvent('onclick',function(event){
  alert(event.type);  //click
})


//法二：作为window的一个属性

oBn.attachEvent('onclick',function(){
  var event = window.event;
  alert(event.type);  //click
})

html特性指定

同DOM中事件对象处理一致，用event变量来对待处理。

共有属性

属性 类型 意义
cancelBubble Boolean 默认为false，若为true则取消事件冒泡
returnValue Boolean 默认为true，若为false则取消事件的 默认行为
srcElement Element 事件目标
type string 事件类型

srcElement
由于这里的作用域要根据指定事件的方法来定，所以事件中this不再始终是事件目标，所以要用srcElement属性来判断

跨浏览器事件对象

总结

总的来说，DOM0级和DOM1级事件处理程序采用传参来获取event对象；因为IE事件处理程序的对象是window，所以，IE事件对象可以当做是window的一个属性来存在；html特性指定将event当做是定义好的event变量使用
