SEO基础

发表于 2018-03-07

内容

三大标签

概念介绍

1.源代码

源代码简单来理解就是浏览器解释网页给搜索引擎看的“语言”，搜索引擎看到的东西和我妈看到的不一样。

2.title标签

<title>搜狐-中国最大的门户网站</title>

3.keywords标签

<meta name="keywords" content="搜狐，门户网站，新媒体，网络媒体，新闻，财经，体育，娱乐..."/>

4.description标签

<meta name="Description" content="搜狐是全球最大的中文门户网站"/>

作用

1.title标签：

这是一个页面中最重要的标签之一，搜索引擎判断页面属性，最为重要的参考。

2.keywords标签

它是关键词的意思，但只是放置关键词的地方，以前这个标签很重要，现在已经被搜索引擎放弃，甚至成为判断网站是否过渡优化的依据。keywords标签建议不写任何东西。

3.description标签

它自身对SEO的影响可以忽略不计，但是会直接关系到用户体验和点击率，而这些确实会影响到SEO。

关键词

概念

SEO真正的价值是带来定向的客户，定向客户是通过搜索一些关键词来到我们网站的，这些客户要搜索的词语就是关键词。

分类

1.主关键词

我们用最高权重页面（比如首页）去优化的词我们叫做主关键词，使我们将来的网站客户客户搜索量最大、商业价值也最大的词。比如：“手机”

2.长尾关键词

更主关键词有关，但搜索量没有它大的一些词，一般用来做网页优化。比如：“诺基亚手机在北京哪里有卖？”

注意：主关键词和长尾关键词都是相对的。比如“长城汽车”这个词，在长城汽车官网就当做主关键词来优化，但是它对于汽车之家这个网站就是一个长尾关键词。

文章

文章主要由文章标题和文章内容组成。

文章标题

  1. 给人看的
  2. 给SEO搜索引擎看的，在源代码中的部分

文章内容

  1. 给人看的
  2. 给SEO搜索引擎看的，为了让给搜索引擎更容易抓取，排名更高。文章是我们优化网站的一个必要手段，在“内容为王”的基础上，对于搜索引擎来讲文章就是最终要的。

分为 

1). 原创：搜索引擎收录过的
2). 伪原创：复制粘贴原创的文章

用户体验

在上述讲文章对于搜索引擎的重要性，但可想而知，搜索引擎是看不懂文章的，所以它只能从用户行为来推断网页的质量。
举例：

  1. 用户在网页上停留的时间
  2. 点击该网页的数量
  3. 用户的收藏数量
  4. 别的网页对该网页的推荐数量

网页的质量提高，对于该网站的信任度也就提高了。

链接

概念

类如 https://www.baidu.com

反向链接

A页面上有B页面的链接，就交A给B做了一个反向链接

任何一个链接都是某一个页面对另一个页面的反向链接。任何一个超链接都是对另一个页面的信任投票。

锚文本

在文本上加上超链接，比如：百度

这就是对另一个页面最好的信任投票，直接告诉了搜索引擎这个链接的属性。

URL超链接

比如：https://www.baidu.com

url就是网站的地址

文本链接

比如： https://www.baidu.com

搜索引擎是根据(超链接周围文字)来判断该超链接是对谁的一个信任

分类

内链

是反向链接的一种重要形式（首页和内页）

http://yimi.com/      首页
http://yimi.com/seo   内页

作用：

  1. 它可以上搜索引擎更多的抓取我们的页面，以及可以让网页之间相互进行投票
  2. 内链具备相关、稳定的特点

但一个网页总共内链不易过多

外链

是反向链接的一种形式，主要指在别的网站网页上对我们网站王页进行的信任投票。

作用：

  1. 反映了该网站的受欢迎程度：在同等条件下，外链数越多，排名和稳定性越好。同等条件下，越是相关的页面越有价值。
  2. 外链是反向链接的重要组成部分：是搜索引擎对我们网站权威性的评价。

内容为王，外链为皇

应用

网站地图

搜索引擎看网站就是看连接，所以由我们网站上所有网页组成的“链接地图”就是我们的网站地图

作用：

增加搜索引擎查阅我们网站的效率，可以显著提升我们网站在搜索引擎面前的曝光率，增加搜索引擎对我们网站的到访次数。类似有引导的作用

位置：

举例：百度>淘宝>最底部，网站地图

死链接

打不开的链接，比如之前一个可以打开的页面被删除了

html

html基础

概念

HTML（Hypertext Markup Language）是用来描述网页的一种标记性语言，是一种规范，一种标准，它通过标记符号还显示网页中的各个部分。

  1. HTML指超文本标记语言
  2. HTML不是一种编程语言，而是一直标记语言
  3. 利用一套标记符号（标签），来描述网页
  4. 组成文件的后缀为.html或.htm

语法规则

  1. <元素名>内容</元素名>
  2. <元素名/>

网页结构

<!DOCTYPE>
<html>
<head>
<title>...</title>
</head>
<body>
<p>...</p>
</boody>
</html>

静态化

静态网页和动态网页

  1. 静态网页：服务器上存在的页面

  2. 动态网页：通过执行asp、php、jsp、.net等程序生成客户端页面代码的网页 

    关键词：架构与内容分离、磨具、转换生成

过程：数据库>动态网页>静态网页

静态网页和动态网页的优劣势：

静态网页

优势： 访问速度快，可以跨平台和服务器

劣势： 占用空间资源多，页面多，难以管理，修改不方便

动态网页

优势： 通过用户请求返回数据库页面，网站易于维护

劣势： 页面不真实存在，对搜索引擎不友好

静态化作用

  1. 有利于搜索引擎抓取（动态页面静态化）
  2. 提高程序性能（静态页面打开速度快）

静态化处理方式

  1. 插件静态化
  2. 代码静态化
  3. HttpWebRequest返回页面重定向法

Nofollow

概念：nofollow是一个html标签的属性。这个标签的意义是告诉搜索引擎“不要追踪此网页上的链接”或“不要追踪此特定的链接”

<a rel="nofollow" href="/index.html"></a>

作用：是站长管理和搜索引擎之间对待一个页面中链接或特定链接的方式。

解读：比如A页面中有一个链接指向B页面，如果我给这个链接加上nofolloe属性，那么A页面的这个链接就不属于B页面的反向链接，并且也不会分散A页面的权重。

权重传递概念–百度知道

网页地址

  1. 绝对地址： 文件在网络或本地的绝对位置，具有唯一性
  2. 相对地址：就是被链接文件相对于当前页面的 地址
  3. SEO考虑：
    含有绝对地址的网页对SEO优化更有利。

对于未做301重定向的网站，不利于搜索引擎确定主域名

相对地址为他人镜像网站提供方便

绝对地址用于网址规范以及首选域问题的考虑

技术

http状态码

http状态码是服务器与客户端（包括用户和爬虫）交流信息的语言
  1. 定义：我们访问任何一个网页的时候，服务器都会生成网站日志。在日志里会有3为数字的代码，这3为数字就记录了我们的访问动作。这3位数字代码就叫做http状态码。

  2. 常见形式：

    2XX  请求已被服务器成功接收
     200 服务器已成功处理请求并提供了请求网页，正常访问
     204 服务器成功处理请求，但并无任何返回
3XX  为完成请求，客户需进一步细化请求（重定向）
     301 请求的网页已永久被移动到新位置
     302 请求的网页临时被移到新位置
     304 该网页自上次请求过后再也没有更新
4XX  客户错误
     400 服务器不理解请求语法  
     403 服务器拒绝请求
     404 服务器找不到请求的页面，错误页面
5XX  服务器的错误
     500 服务器遇到错误，无法完成请求
     503 服务器当前无法使用（超载或停机维护）

  3. 对SEO的意义：

    通过状态码可以查询搜索引擎爬虫在你网站的爬取情况。
通过状态码可以看出网站的情况。

爬虫

  1. 定义：网络爬虫（也叫网络蜘蛛），是一种自动获取网页内容的程序，爬虫抓取的网页将会被搜索引擎系统存储，进行一定的分析，过滤，并进行索引，以便以后的用户能够查询到这个页面。这个获取信息的程序就佳作爬虫。
  2. 爬虫与搜索引擎的关系：爬虫为搜索引擎收集内容，搜索引擎来展示。

  3. 工作流程：爬虫通过漫游的方式进行抓取。爬虫爬到一个页面后，看到一个链接，然后顺着那个链接又爬到另一个页面，爬虫是不停的从一个页面跳到另一个页面的，它一边下载这个页面，一边提取这个网页中的链接，那个页面上所有的链接都放到一个公共的“待抓取列表”里。而且爬虫有个特点，就是在他访问你网站之前，不去做判断你这个网页本身是怎么样的，不管页面内容判断就抓取，但是会有优先级的划分，尽量不抓取重复的内容，劲量抓取重要的内容 （比如网站的公共部分）。所搜引擎同时会派出多个爬虫进行多线程的抓取，所有被爬虫抓取的网页将会被系统存储，进行一定的分析、过滤（去重），并建立索引，以便以后查询和检索。

  4. 分类：根据搜索引擎来分类
    百度爬虫：Baiduspider百度蜘蛛
    谷歌爬虫：Googlebot谷歌机器人
    soso爬虫：sosospider

搜索引擎入口

  1. 定义：刚刚建立好的网站，向搜索引擎提交一下，让搜索引擎知道你的网站，让搜索引擎能够访问到你的网站，能自动收录网页（经过一系列审核）。
  2. 提交：搜索“搜索引擎提交各大入口”

权重

搜索引擎对一个网站的任性度。测量不错具体数据。

PR值

PageRank（网页级别）

  1. 踏实谷歌用于表示网页的等级、重要性用的一种方法，是谷歌何亮一个网站好坏的重要指标之一。
  2. 可以测量。搜索“站长工具>PR值测量”即可查询

robots

  1. 定义：一个协议，告诉搜索引擎爬虫什么可以抓取，什么不可以
  2. 位置：一个网站文档里的robots.txt
  3. 功能：屏蔽无用信息，节省服务器的贷款资源，提升收录。
  4. 查询：https://www.taobao.com/robots.txt

  5. 生成：

    法一：新建txt文件，编写格式如下：
User-agent:  Baiduspider                //禁止百度爬虫抓取
Allow:  /article                        //允许抓取改文件
Allow:  /oshtml   
Disallow:  /product/                   //不允许抓取改文件
Disallow:  /                           //不允许抓取所有文件

    法二：（百度搜索）站长工具>网站分析>robots
