跨域请求

含义

  1. 跨域是一个域下的文档或脚本试图去访问另一个域下的资源。
  2. 它是由浏览器的同源政策施加的安全限制

跨域的解决方案

  1. 通过jsonp跨域
  2. document.domain+iframe跨域
  3. location.hash+iframe跨域
  4. window.name+iframe跨域
  5. postMessage跨域
  6. 跨域资源共享（CORS）
  7. nginx代理跨域
  8. nodeis中间件代理跨域
  9. WebSocke协议跨域

jsonp

  1. 实现原理：类如标签可以写入html文件里，添加src属性引入不同域下的文档渲染网页。基于此原理，我们可以动态创建script标签，在添加一个带参网址实现跨区请求。
  2. 原生代码如下：
<script>
//创建标签
var script=document.createElement("script");
script.type='text/javascript';
//传参并指定成功时服务器的·回调函数
script.src='http://www.domain2.com:8080/login?user=admin&callback=onBack';
//动态添加标签
document.head.appendChild(scipt);
//回调执行函数
function onBack(res){
  alert(JSON.stringify(res));
}
</script>

成功时服务端返回如下：

onBack({"status"：true，"user":"admin"})

3.jquery代码实现如下：

<script>
function onBack(res){
  alert(JSON.stringify(res));
}

$.ajax({
url:'http://www.domain2.com:8080/login',
type:'get',
dataType:'jsonp',
jsonpCallback:'onBack',//自定义回调函数，或者用ajax的success属性
data:{}
});
</script>

4.vue.js代码如下：

this.$http.jsonp('http://www.domain2.com:8080/login', {
params: {},
jsonp: 'onBack'
}).then((res) => {
console.log(res); 
})

后端node.js代码示例：

var querystring = require('querystring');
var http = require('http');
var server = http.createServer();

server.on('request', function(req, res) {
   var params = qs.parse(req.url.split('?')[1]);
   var fn = params.callback;

   // jsonp返回设置
   res.writeHead(200, { 'Content-Type': 'text/javascript' });
   res.write(fn + '(' + JSON.stringify(params) + ')');

   res.end();
});

server.listen('8080');
console.log('Server is running at port 8080...');
