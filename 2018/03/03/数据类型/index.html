数据类型

发表于 2018-03-03

5种简单的数据类型（基本数据类型）：Undefined,Null,Boolean,String,Number
1种数据类型（引用数据类型）：Object

typeof操作符

undefined———(typeof)————undefined（没有定义改变量，或定义了但没有初始化）
null————————————-object(特殊的null被认为shi一个空的对象引用)
boolean
number
string
function
object———————————-object

Undefined

使用var声明但未对其初始化

注：当typeof 变量为undefined时，表示改变量未被声明，而不是声明后未初始化

Null

null指表示一个空对象指针，故
alert(typeof null);//object

实际上，undefined的值是派生自null的，故
alert(null==undefined);//true
但 alert(null===undefined);//true

有意保存对象的变量而没有对象的值，可以讲对象定义为null值，以便以后判断其是否有值

Boolean

字面值 true 和 false

将一个值转换为布尔值： Boolean（）
Boolean（非空字符串）//true Boolean（“空格 ”）//true Boolean（“”）//false
Boolean（非0数字）//true Boolean（0）//false Boolean（NaN）//false
Boolean(除null外的对象) //true Boolean(null) //false
Boolean(undefined)//false

Number

十进制
八进制 0+（0~7）
十六进制 0x+（0~9/A~F）
e指数型的数值，3.175e7机器识别最终为31750000
在数值超出规定无穷大限制是表示为Infinity或-Infimity((Infinity是不能参与计算的数值)
判断一个数值是否是有穷性的，用 isFinite()函数
Number.NEGATIVE_INFINITY (-Infinity)
Number.POSITIVE_INFITY (Infinity)
NaN (not a number)反映本身要返回数值的操作数未返回数值的情况
alert（number/不是数字的值）;//返回NaN
alert（NaN/10）;//NaN NaN参与的运算，返回都是NaN
alert(NaN==NaN);//false NaN与任何数值都不相等，包括本身
isNaN()函数，将拿到的值先会转换成数值，成功返回false，如
alert（isNaN(NaN)）;//true
alert（isNaN(“10”)）;//false （可以被转换成10）
alert(isNaN(true)); //false (可以被转换成1)
alert(isNaN(‘blue’)); //true
其过程可以理解为是，先执行valueof(),确定返回的值是否可以转
换成数字，若不能，基于这个返回值再用toString（）测试返回值

将非数值转换为数值：

  1. Number( )
    规则如下：
    如果是Boolean，true返回1；false返回0
如果是数字，简单地传入传出
如果是null，返回0
如果是undefined，返回NaN
如果是字符串， Number（‘123’）//123
                         Number（‘0123’）//123
                         Number('1.1')//1.1
                         Number('0xA')//1.0  十六进制转十进制
                         Number('')//0           Number('空格')//0
                         除上述外任何其他为NaN
    如果是对象， Number({ })//NaN
  2. parseInt( )
    忽略字符串前面的空格，直到找到第一个非空格字符；直到解析到非数字字符结束。
    如果第一个非空字符不是数字或者是负号，会返回NaN
    parseInt(' ')//NaN
parseInt(' -')//NaN
parseInt('123erfd ')//123
parseInt(' 11.4')//11        parseInt(11.4)//11
parseInt('0xA')//10          parseInt('0xA'，16)//10       parseInt('A'，16)//10   十六进制（推荐使用基数 ）
parseInt(' 070')//70
  3. parseFloat( )
    prseFloat('123dew')//123
prseFloat('123.2')//123.2
prseFloat('123.2.43')//123.2
prseFloat('0xA')//0
prseFloat('092.3')//92.3
prseFloat('1.12e5')//112000

eg var s=’duyi’; ,在这里单引号双引号无区别

  1. 字符字面量(转义字符)
    \n 换行
    \t制表
    \b退格
    \r回车
    \斜杠
    \’单引号
    \”双引号
  2. 转换为字符串
    toString()
       返回该数值的一个副本，**null和undefined没有这个方法**
   var s=11；
 *无参*console.log(s.toString());//'11'
*带参*console.log(s.toString(2));//'1011'

Object

引用类型是一种数据结构，将数据和功能结合起来，常被称为类，但并不妥当；也常被称为对象定义，描述一类对象所具有的属性和方法。
引用类型的值（对象）是引用类型的一个实例。

总结：对象是某个特定引用类型的实例。新对象使用new操作符后跟一个构造函数来创建。构造函数本省是一个函数。只不过该函数出于创建新对象的目的而建立。

创建实例方法

  1. var obj=new Object();
    2.对象字面量
    var obj={
    name:'duyi',
age:'18'
    };

或
var obj={};
obj.name=’duyi’;
obj.age=’18’;

同样，属性名也能用字符串 ,5自动转化为字符串
var obj={
‘name’:’duyi’,
‘age’:’18’,
5:2
};

每个object实例都具有的属性和方法

  • constructor：保存着用于创建当前对象的函数
  • hasOwnproperty(propertyName):用于检测给定的属性在当前对象实例中（而不是实例原型中）是否存在
    eg: object.hasOwnproperty("name");
  • isprototypeOf(object):用于检测传入的对象是否是当前对象的原型
  • propertyIsEnumerable（propertyName）：用于检查给定的属性能否使用for-in语句来枚举
  • toLocaleString（）:返回对象的字符串表示，与执行环境的地区对应
  • toString（）：返回对象的字符串表示
  • valueOf（）：返回对象的字符串，数字或布尔值表示

访问属性

var obj={
   name:'duyi',
   age:'18'
};

alert(obj.name);
alert(obj['name']);

创建数组

1.构造函数

2.数组字面量
var arr=[];
var arr=[“red”,”yellow”,”blue”];

eg: var arr1=[];
var arr2=[“1”,”2”,”3”];
alert(arr1.length);//0
alert(arr2.length);//3
alert(arr1[0]);//undefined
arr2.length=4;
alert(arr2[3]);//undefined
arr1[0]=’12dx’;
alert(arr1[0]);//‘12dx’
arr2[99]=’98’;
alert(arr2.length);//100
